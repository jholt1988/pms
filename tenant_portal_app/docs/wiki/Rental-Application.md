# Rental Application

The Rental Application feature allows prospective tenants to apply for a rental property, and property managers to view and manage rental applications.

## Applying for a Rental (Prospective Tenants)

To apply for a rental property, navigate to the "Rental Application" page. You will see a form where you can enter your personal information, employment details, and previous address. After you submit the form, your application will be sent to the property manager.

## Managing Applications (Property Managers)

To view and manage rental applications, navigate to the "Applications" page. You will see a list of all rental applications. You can approve or reject an application.
